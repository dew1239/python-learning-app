# python-learning-app
# Python Learning App (Educational)

## Features
- Username gate ก่อนเข้าใช้งาน
- Lessons + Inline Playground
- Quiz ท้ายบท + บันทึกคะแนน
- Dashboard สรุปผล
- Corner Chat ผู้ช่วยสอน (Gemini)

## Run locally
pip install -r requirements.txt
streamlit run app.py

## Notes
- หากไม่ตั้งค่า GEMINI_API_KEY/GOOGLE_API_KEY ระบบแชทอาจใช้งานไม่ได้
